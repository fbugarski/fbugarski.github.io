Confidential Computing Solutions
for Cloud, IoT and Telecom

Ultraviolet Consult is a cloud computing company oriented towards secure and confidential computing technologies in the domain of telecom, cloud and IoT systems.

Confidential Computing

Confidential Computing is a concept in which encrypted data can be processed in memory. Besides traditional techniques of keeping data safe "at rest" (database encryption) and "in transit" (TLS and other transport encryption mechnisms), UV is keeping data safe "in use" as well, leveraging on a new security model called Confidential Computing.

UV has a long tradition and expertize in telecom domain. Based on this experience, UV has built secure solutions for 5G, IoT and LPWAN connectivity and analytics. UV IoT cloud platform helps connect and manage various types of sensors and actuators via different connectivity networks: LTE, NB-IoT, LoRaWAN and other. Platform is provided as a whitelabel and "as-a-service" (managed SaaS) for the customers which can than operate it and manage their IoT systems. UV provides all related services, from maintenance and support, to buidling various IoT solutions on top.

Secure AI/ML

UV develops secure and privacy-preserving leveraging on Confidential Computing techniques and data acquisition and protection technologies coupled with secure connectivity in telecom and IoT.

UV Secure AI/ML solution is based on an orchestration layer to safely deploy AI/ML algorithms in containers over the VMs with secure TEE HW extensions. This way both data nor algorithms are never decrypted and can stay private and protected in use.

Privacy-preserving feature is often referred to as a "holly grail of AI", and opens novel use-cases, like Secure Multy-Party Computation where multiple companies and enterprises can securely share data and execute analytics on combined data sets.
