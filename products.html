PRODUCTS

Ultraviolet Consult is a cloud computing company oriented towards secure and confidential computing technologies in the domain of telecom, cloud and IoT systems.

Teleflux - Secure IoT Platform

Teleflux is a carrier-grade multi-protocol unified IoT platorm. It is cloud-based and can be deployed as a whitelabel in private, public or hybrid clouds. Teleflux is based on microservice architecture and each component is individually clusterable and scalabe. It is falut-tolerant and highly-available, robust and can be scaled to manage millions of devices simultaneously.

Teleflux is deployed as a managed service, a turn-key solution. Integration APIs are provided and it is easy to extend with new protocols, connectors, enterprise message busses and databases. IN that sense, Teleflux is ideal for CSP, netowork and service operators and the enterpise.

CoCoS.ai - Privacy-Preserving AI/ML orchestration

CoCoS.ai is a SaaS cloud system dedicated for privacy-preserving AI/ML, i.e. execution of model training and algorithm inference on confidential data sets. Privacy-preserving is considered as a “holy grail” of AI. It opens many possibilities, among which a collaborative trustworthy AI. CoCoS.ai is a distributed, microservice based solution in the cloud that enables confidential and privacy-preserving AI/ML while enabling following features:

1.Data Scientist pipelines with UI
2.User and Key Management
3.Distributed computation orchestration over TEE-enabled machines
4.Programmable TEE environments (novel protocols)
5. brokering while adhering to most recent IETF Open Trust Protocol standards
6.Providing API for programmable manipulation with the platform

Final product enables Data Scientists to train AI/ML models on confidential data that is never revealed, and can be used for Secure Multi-Party Computation (SMC).
